# spi_25aa02e48
Codigo C++ para un Raspberry Pi Zero W2 con un IC 25AA02E48
comunicado por SPI 

el codigo funciona correctamente con la libreria spi.cc y spi.hpp

el puerto en el raspberry Pi es el MODO_0




pin 19 SDO
pin 22 SDI
pin 23 SCK
pin 24 CS
